## 📱 アプリの特徴

### ✨ 機能
- **カメラ撮影**: デバイスのカメラから画像をリアルタイムで取得
- **AI分析**: Google Generative AI (Gemini 2.0 Flash) を使用した高速な画像分析
- **自然言語処理**: 日本語での質問に対応
- **撮り直し機能**: 不満足な撮影画像は簡単に再撮影可能

### 🎨 UI デザイン
- **Tailwind CSS**: モダンで反応性の高いレスポンシブ デザイン
- **グラデーション**: 視覚的に魅力的なヘッダーとカメラセクション
- **ロード状態**: 回答生成時のスピナーアニメーション
- **エラーハンドリング**: ユーザーフレンドリーなエラーメッセージ表示

### 📦 ファイル構成

```
/content/
├── main.py              # Flask サーバー (port=3000)
├── requirements.txt     # Python 依存関係
└── index.html          # Tailwind + カメラUI
```

### 🚀 デプロイ方法

**Google Cloud Functions へデプロイ可能:**
```bash
gcloud functions deploy main \
  --runtime python311 \
  --trigger-http \
  --entry-point main \
  --set-env-vars OPENAI_API_KEY=<your-api-key>
```

### 🔑 環境変数
アプリは `OPENAI_API_KEY` 環境変数を使用します。Google Generative AI の API キーを設定してください。

### 📝 使い方
1. 「カメラ起動」ボタンをクリック
2. 「撮影」ボタンで画像をキャプチャ
3. 質問を入力して「質問」ボタンを送信
4. AI が画像について回答します
5. 「撮り直し」で再度チャレンジ

すべてのコードは `/content` ルートディレクトリに配置されており、Cloud Functions にそのまますぐデプロイできます！
